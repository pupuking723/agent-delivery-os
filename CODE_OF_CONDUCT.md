# Code of Conduct

## Our Standard

This repository exists to help people build and operate delivery systems with agents.
We expect collaboration to stay practical, respectful, and technically grounded.

Examples of behavior that contributes to a healthy environment:

- giving clear and actionable feedback
- challenging ideas without attacking people
- sharing context, assumptions, and tradeoffs
- respecting different experience levels and backgrounds
- focusing discussions on improving the work

Examples of unacceptable behavior:

- harassment, threats, or personal attacks
- insulting or dismissive language
- deliberate disruption of discussions or reviews
- publishing private or sensitive information without permission
- bad-faith arguments that block progress without technical substance

## Scope

This code of conduct applies to:

- issues
- pull requests
- discussions
- release conversations
- other repository-linked collaboration spaces

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.
They may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the repository maintainer through the support path in [SUPPORT.md](/Users/mac/Desktop/other/全栈/SUPPORT.md).

If the report includes a potential security issue, use [SECURITY.md](/Users/mac/Desktop/other/全栈/SECURITY.md) instead of filing a public issue.
