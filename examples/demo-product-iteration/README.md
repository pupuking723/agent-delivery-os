# Demo Flow: Product Iteration

这是一套完整的演练包，用来展示 Delivery OS 在 `Product Iteration` 模式下的真实流转结果。

它不是模板，而是示例产物集合。

## 包含什么

- issue form 示例
- 自动 kickoff 评论示例
- 自动 artifact checklist 评论示例
- 生成出的 workspace 路径示例
- PR body 示例
- merge 后的 PR / issue 提醒示例
- iteration-log 补写示例
