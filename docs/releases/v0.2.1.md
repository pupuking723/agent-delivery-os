# Release v0.2.1

这是 `agent-delivery-os` 的一次上手体验优化版本。

## 这版做了什么

这一版不新增复杂能力，重点是降低第一次使用的学习成本。

## 这版包含什么

- 新增：
  - [docs/quickstart.md](/Users/mac/Desktop/other/全栈/docs/quickstart.md)
  - [docs/quickstart.en.md](/Users/mac/Desktop/other/全栈/docs/quickstart.en.md)
- README 首页改成“先记住 3 个命令”的入口
- 新增 `pnpm health:check`
- 保留 `doctor` 脚本，但不再把 `pnpm doctor` 作为默认对外命令
- `doctor` 自检规则已纳入 `health:check`

## 为什么要发这一版

前一版已经把初始化和自检能力做齐了，但第一次上手时暴露的命令和文档入口还是过多。
这一版的目标是把新手默认心智压缩成：

```bash
pnpm bootstrap:all -- --project "新项目名" --repo-name "new-product-repo" --repo <owner/repo>
pnpm kickoff --title "需求标题" --summary "需求说明"
pnpm health:check -- --repo <owner/repo>
```

## 当前推荐用法

先初始化仓库，再发起需求，最后做健康检查。

## 下一步方向

- 增加更细的业务仓库检查规则
- 用真实业务仓库继续验证新手入口是否足够简单
