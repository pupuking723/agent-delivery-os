# Release v0.1.1

这是 `agent-delivery-os` 的第一次公开包装更新。

## 这版做了什么

这一版不改三模式工作流本身，重点是把仓库补成一个更完整的公开开源项目。

## 这版包含什么

- README 首页增强：
  - release badge
  - license badge
  - 更清晰的定位、场景和入口说明
- 社区健康文件：
  - [SECURITY.md](/Users/mac/Desktop/other/全栈/SECURITY.md)
  - [SUPPORT.md](/Users/mac/Desktop/other/全栈/SUPPORT.md)
  - [CODE_OF_CONDUCT.md](/Users/mac/Desktop/other/全栈/CODE_OF_CONDUCT.md)
- GitHub 展示面优化：
  - 仓库 homepage 指向 playbook
  - topics 已补齐，方便检索和外部展示

## 为什么发这一版

`v0.1.0` 已经证明这套系统能跑通模板、脚本和 GitHub 自动化。
`v0.1.1` 的目标是把它从“可用仓库”再往前推一步，变成“更适合公开复用和分享的项目”。

## 当前状态

到这一版为止，仓库已经同时具备：

- 可执行的交付系统骨架
- 可展示的 demo 和 playbook
- 可公开复用的许可证
- 基本完整的社区健康与支持入口

## 下一步方向

- 继续把这套系统接入真实业务仓库
- 补更多真实样本和对外展示材料
- 在后续版本继续增强 release 和 feedback 自动化
